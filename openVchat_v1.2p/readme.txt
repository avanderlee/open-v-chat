Readme openVchat v1.2p

This file contain the web files and database files.
To configure your own server, you can modify localhost.sql (in order to change database name)
By default, the database named openvchat.
To change configuration on the web files, modify openvchat/util/utility.php
change the getter method for hostname, username, password, and database name.

If you have any question, feel free to send mail to andre_flea@yahoo.com

ps: please use localhost.sql if this is your first time to install openVchat on your server.
    Use upgrade_from_1.1p.sql if there's already exists openVchat v1.1p on your server
